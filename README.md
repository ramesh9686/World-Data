# 🌍 World Explorer — Interactive World Map

An interactive Angular application that lets users explore countries around the world by clicking on an SVG world map. Selecting a country fetches live data from the **World Bank API** and displays useful information including the country's capital, region, income level, latitude, and longitude.

---

## ✨ Features

* 🌍 Interactive SVG world map
* 🖱️ Click any country to view its details
* 📡 Live country information from the World Bank API
* 📱 Responsive design for desktop and tablet devices
* 🎨 Smooth hover and selection animations
* ⚡ Fast, lightweight Angular application

## 🚀 Getting Started

### Prerequisites

* Node.js 18 or later
* npm 9 or later

### Clone the Repository

```bash
git clone https://github.com/ramesh9686/World-Data.git
cd World-Data
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
ng serve
```

Open:

```
http://localhost:4200
```

The application automatically reloads whenever you modify the source code.

---

## 📦 Build for Production

```bash
ng build
```

The production build will be generated inside the `dist/` directory.

---

## 🧪 Run Tests

```bash
ng test
```

## 🌐 Data Source

This project uses the **World Bank Country API** to retrieve live country information.

API Endpoint:

```
https://api.worldbank.org/v2/country/
```
